1. se crea un repositorio se hace mediante git init
2. se observa el estado del repositorio con git status
3. se añade los archivos que faltan con git add .
4. se realiza un commit con git commit -m y se añade un mensaje
5. si te pide autenticarte se hace mediante git config -- global user.name username
6. lugo git config --global user.email email
7. se debe dirigirse a mis repositorios de github y crear uno nuevo
8. se copia el quick setup en ssh
9. se escribe git remote add origin y se pega el quick setup que se copio
10. luego se escribe git push origin master
